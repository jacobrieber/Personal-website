Operation warp speed

November 2021 - March 2022

Exploration into all things web development, producing a variety of projects from static web-based games to full-stack blogging platforms

Technology: HTML, CSS, JavaScript, EJS, Express, Node.js, MongoDB

Project overview

“Operation warp speed.” It’s a dramatic title — I’ll admit it, but it feels appropriate.

This past half of a year has been a busy one. I’ve been working in a more intense group at work and have regularly been putting in 15+ hours per day. But you know what? Sometimes you need a bit more action in your life, so I reacted like a sane human being and decided to teach myself how to code on top of that. And - drum roll - it's been a ton of fun! Honestly, my main motivation here was I wanted to start making things again. In high school, I used to love designing sets and turning something I envisioned into a reality. Coding felt like a way I could start doing that again.

I took one coding class in college, but I had a lot of catching up to do if I wanted to be able to make anything decent, so on my weekends and in spare moments during the work week, I practiced coding and built out a few projects to try and nail the basics down. I’ve felt like the world’s most, believable batman — a corporate man by day and a mean, green coding machine by night.

To start my coding adventure, I decided to focus my initial efforts on web development. Websites are extremely scalable, have tons of use cases, and are really tangible. Below are a select projects I've done throughout this coding ramp-up period. These projects range from static web-based games to full stack blogging platforms.

Sample Web Dev Projects

Journal Website

Full-stack blogging platform that allows the user to write and delete blog posts. Posts and their associated URLs are dynamically generated and are stored externally via a MongoDB server.

Technology: HTML, CSS, JavaScript, EJS, Express, Node.js, MongoDB

To-Do List Application

Full-stack to-do list application that let's you create multiple lists with unique, auto-generated URLs. These lists are stored externally via a MongoDB server.

Technology: HTML, CSS, JavaScript, EJS, Express, Node.js, MongoDB

Newsletter Sign-up Page

An experiment to get familiar with using APIs. This webpage collects contact information and stores this infromation on MailChimp.

Technology: HTML, CSS, JavaScript, Node.js, MailChimp API

The Simon Game

Static website that replicates the Simon Game.

Technology: HTML, CSS, JavaScript

Drum Kit

Playable drum kit website built to explore basic JavaScript functionalities

Technology: HTML, CSS, JavaScript
